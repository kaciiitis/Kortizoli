# Kortizoli

Vienkārša Python grafiskā lietotne stresa mazināšanai.

## Par lietotni
“Kortizoli” ir izveidota kā neliels palīgs stresa situācijās, kas palīdz nomierināties, izmantojot pozitīvas afirmācijas un elpošanas vingrinājumus.

## Funkcionalitāte
- Pozitīvu afirmāciju parādīšana
- Elpošanas vingrinājums
- Vienkārša grafiskā lietotāja saskarne (GUI)

## Izmantotās tehnoloģijas
- Python
- Tkinter

## Programmas palaišana
python main.py
